# 安全政策 (Security Policy)

我們對 **ZJ Card 奇蹟卡** 的安全性非常重視。如果您發現任何安全漏洞，請透過以下流程與我們聯繫，我們將優先處理。

## 支援的版本 (Supported Versions)

目前我們僅針對最新的主分支代碼提供安全更新。

| 版本 | 是否支援安全更新 |
| :--- | :--- |
| **Latest (main)** | ✅ 支援 |
| < 2.x.x | ❌ 不支援 |

## 如何回報漏洞 (Reporting a Vulnerability)

**請勿直接在 GitHub 上建立公開的 Issue 來回報漏洞。**

為了保護使用者的安全性，請透過 **LINE 官方帳號** 私訊我們進行私人回報：

👉 **[JZC Design 官方 LINE](https://lin.ee/tU0a6pR)**

在回報時，請儘可能提供以下資訊：
1. **漏洞描述**：該漏洞是什麼，以及它如何影響系統？
2. **重現步驟**：教學我們如何重現該問題。
3. **影響範圍**：哪些功能或資料可能受到影響。

## 回應時程 (Response Process)

- 我們將在收到回報後的 **48 小時內** 給予初步回應。
- 確認漏洞後，我們將在修復完成前保持溝通，並在修復後發布安全更新。

感謝您協助我們維護 **ZJ Card 奇蹟卡** 的安全性！
